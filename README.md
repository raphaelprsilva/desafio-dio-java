# Desafio de código - Java

Neste desafio, o objetivo é criar uma simulação de um banco.

Nele, pude trabalhar e entender alguns conceitos fundamentais do Java, tais como:

- Programação Orientada a Objetos
- Herança
- Encapsulamento
- Polimorfismo
- Interfaces
- Classes Abstratas
- Getters e Setters
- Constructor
